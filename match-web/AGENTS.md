# Produto

Este é o **match-web** — o **painel do organizador** do MatchIndustrial365 (aprovar/publicar
expositores, posicionar estandes no croqui, gerir dados do evento). Faz parte do
MatchIndustrial365 (comunidade B2B "365", o ano todo) — **não** confundir com o
**ExpoIndustrialApp** (app do evento, durante a feira).

⚠️ Compartilha o MESMO Firestore (`movie-app-ddda3`) que os outros apps — reaproveite as
coleções já existentes. No Next.js as chaves Firebase usam prefixo `NEXT_PUBLIC_*`.
